# InvoiceForge

Free, client-side invoice generator. No account required, no watermarks, no data leaving the browser.

## Features
- 4 professional invoice templates
- Logo upload (remembered in localStorage)
- Saved client list with CSV import
- Rate sheet for quick line item reuse
- Tax, discount, shipping, deposit calculations
- PDF download — 100% client-side
- Auto-saves to browser localStorage
- 10 currency symbols

## Live Site
[invoiceforge.com](https://invoiceforge.com)

## Tech Stack
- Vanilla HTML/CSS/JS
- html2canvas + jsPDF for PDF generation
- localStorage for all persistence
- Cloudflare Pages hosting
